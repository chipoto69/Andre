## Skip Functionality

All tour screens support optional skip button:

```swift
ListsTabTourScreen(
    onContinue: { /* Next screen */ },
    onSkip: { /* Jump to app */ }  // Optional
)
```

Toolbar skip button appears in top-right when `onSkip` is provided.
